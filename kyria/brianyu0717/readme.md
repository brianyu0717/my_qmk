## FUNC Layer: Function keys, and easy to reach goodies

* ```CG_L/R``` Move Windows Desktops L/R
* ```CS_H / CA_U``` IntelliJ shelf/unshelf
* Dedicated ESC key 

```
,-------------------------------------------.                              ,-------------------------------------------.
|  ESC   | F1   |  F2  | F3   | F4   | F5   |                              | F6   | F7   |  F8  | F9   | F10  |        |
|--------+------+------+------+------+------|                              |------+------+------+------+------+--------|
|        |      |      |  CG_L| CG_R |      |                              |      |      |      | F11  | F12  |        |
|--------+------+------+------+------+------+-------------.  ,-------------+------+------+------+------+------+--------|
|        |      |  CUT | COPY | PASTE|      |TO_RGB|      |  |      |      |      |      |      |      |      |        |
`----------------------+------+------+------+------+------|  |------+------+------+------+------+----------------------'
                       |      | CS_H | CA_U |      |      |  |      |      |      |      |      |
                       |      |      |      |      |      |  |      |      |      |      |      |
                       `----------------------------------'  `----------------------------------'
                                               
```
