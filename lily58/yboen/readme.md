# My lily58 layout

Making a _lite_ version of the kyria layout with some optimizations and improvement.

Goal is to make it work for 34 keys so eventually could easily transition to _end game_ keebs like the ferris.

# Features

## Home row mods

```
 |------+------+------+------+------+------|     |------+------+------+------+------+------|
 |      |   A  |   S  |   D  |   F  |   G  |     |   H  |   J  |   K  |   L  |   ;  |      |
 |      | Lgui | Lalt | Lctl | LSft |      |     |      | Lsft | LCtrl| LAlt | LGui |      |
 |------+------+------+------+------+------|     |------+------+------+------+------+------|
```

## Combos 

* "ERM" for personal email
* "WEM" for work email
* "QWE" for qwerty layer as default
* "WER" for gaming layer as default
